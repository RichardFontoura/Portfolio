Meus Projetos

Aqui você encontrará alguns dos projetos que criei, sendo que cada projeto possui seu próprio estudo de caso.

Bot para Discord:

Uma ferramenta completa para administração de servidores. Com recursos avançados, como banir, expulsar membros e enviar mensagens personalizadas, além de sistemas de criação de embeds, tickets e fichas. Desenvolvido em Python, oferece controle total e organização para um servidor do Discord.

Sistema Flask para Registro e Login de Usuários:

Um aplicativo web desenvolvido em Python com o framework Flask. Permite o registro e login de usuários, armazenando informações em um banco de dados MySQL. Possui páginas de registro, login e logout, além de garantir a segurança das sessões dos usuários. Uma solução prática e segura para gerenciar a autenticação em aplicativos web.
